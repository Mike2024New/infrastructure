# infrastructure

Набор утилит и инструментов, переиспользуемых в различных проектах.

## О проекте

Этот репозиторий содержит общие компоненты, которые используются во многих моих проектах. Вместо того чтобы дублировать
код в каждом проекте, он вынесен сюда и подключается как зависимость.

## Требования

- Python >= 3.12

## Установка

```bash
uv add git+https://github.com/Mike2024New/infrastructure.git
```

## Компоненты (обращение к ним в коде через infrastructure. , например from infrastructure import message_bus)

1. `message_bus` - шина сообщений - общение приложений, логирование.
2. `git_client` - простой git клиент, для автоматизированного получения обновления компонентов.
